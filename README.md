# 命令行有道单词本

ydadd 是命令行版本的有道单词本, 支持添加单词到有道单词本.


## 安装说明

克隆项目或者下载项目文件夹, 把 ydadd 文件中 username，password 改成自己的, 然后使用以下命令安装


```
make
```


## 使用示例

```
$ ydadd love
Success!
$ ydadd 重要的
Success!
```
